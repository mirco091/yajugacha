# YAJU&U NFT GACHA GitHub Pages版

## 使い方
1. このフォルダの中身をGitHubリポジトリにアップロード
2. Settings → Pages → Deploy from branch → main / root
3. `config.js` を編集

## 後で入れるもの
- CA: `config.js` の `ca`
- Discord: `discordUrl`
- Mintページ: `mintUrl`
- BGM: `assets/audio/bgm.mp3` を置き、`bgmFile: 'assets/audio/bgm.mp3'`
- NFT保有チェック: Helius API Key と collectionMint

## 注意
このテンプレートは無料抽選・コミュニティ特典向けのデモです。実際のNFT販売・抽選・権利付与は法務確認してから運用してください。
